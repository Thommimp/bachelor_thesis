Machine learning applied to cellular network data has been widely studied. Most existing work focuses on traffic forecasting using advanced machine learning models [5], while unsupervised learning techniques have also been applied to clustering and identifying regions with high traffic density [6]. In contrast, relatively little attention has been given specifically to the seasonal classification of network towers. One possible reason is that many modern forecasting models are capable of implicitly learning seasonal traffic patterns from the data itself. Furthermore, although simpler statistical approaches such as linear regression are still commonly used as baselines, they may struggle to capture the nonlinearity nature of cellular traffic data compared to more advanced models.
